## なぜ作ろうと思ったのか<br>
１日にやらなくてはいけないことが多く、効率よく管理したいという気持ちがあり、タイマー付きのTodoアプリが欲しいと考えました。
調べたところ、使い勝手の良いアプリはあまりなかったので、自身で作っています。
<br>

・ChatGPT 　 API 機能<br><br>
